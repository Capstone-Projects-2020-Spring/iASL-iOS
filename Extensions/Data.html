extension Data
  • init(copyingBufferOf:)

    Creates a new buffer by copying the buffer pointer of the given array.

    Warning

    The given array’s element type T must be trivial in that it can be copied bit for bit with no indirection or reference-counting operations; otherwise, reinterpreting data from the resulting buffer has undefined behavior.

    Declaration

    Swift

    init<T>(copyingBufferOf array: [T])

    Parameters

    array

    An array with elements of type T.
